# Dotfiles

## Supported Platforms

* Ubuntu
* MacOS

## MacOS

### Dependencies

1. Homebrew
1. Zsh (brew)
1. Vim (brew)
1. git (brew)
1. Monospaced fonts for powerline

### Instructions

`$ ./scripts/bootstrap`

## Ubuntu

### Dependencies

1. git

### Instructions

`$ ./scripts/ubuntu-bootstrap`
